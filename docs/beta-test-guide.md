# つづりび 無償βテスター向け案内

つづりびは、AIが慰めるためのアプリではありません。
結果が出る前の努力や行動の跡を、あとから見つけ直すための自己証拠箱です。

## 試してほしい流れ

1. Homeを見る
   - まず `ここまで` の画面を開きます。
   - まだ記録がない状態でも大丈夫です。

2. 今日のことをひとつ残す
   - `追加` から `ひとつ残す` を開きます。
   - 近い行動を選び、必要なら任意の一言を短い証拠ラベルとして確認します。

3. メモ貼り付けを試す
   - Googleを使わない人は、メモ帳などの短い行を貼り付けて試せます。
   - 日付がない行は、保存前に `今日として残す` / `日付なしで残す` / `保存しない` を選べます。

4. 可能ならGoogleカレンダー確認
   - Google連携は任意です。
   - 予定は勝手に保存されません。
   - 保存前に短いラベルとカテゴリを確認します。

5. 手紙・一文
   - 自分の言葉を短く残せます。
   - AIが文章を生成する画面ではありません。

6. つらい時
   - 元気にするためではなく、ここまでに残っていた跡や一文へ戻る画面です。

7. お問い合わせフォーム
   - Settingsに `お問い合わせフォーム` が表示されている場合は、改善点、不具合、使ってみた感想、連絡したいことを送れます。

## 書かなくてよいもの

お問い合わせフォームには、次の情報を書かなくて大丈夫です。

- 実名
- 企業名
- 具体的な予定名
- Googleカレンダーの詳細
- 個人が特定されるメモ本文

「どこで迷ったか」「どの流れが安心だったか」「また開きたいと思ったか」だけでも十分です。

## まだないもの

無償βには、次の機能は入っていません。

- AI
- 通知
- 課金
- Google裏側定期取得
- Google起動時自動確認
- GitHub / Notion連携
- 月間レポート
- 週はじめのここまで
- 挑戦前レポート
- みんなの一文

## 保存について

現在は、この端末のブラウザ内に保存します。
DB本接続、refresh token保存、裏側定期取得はまだありません。
詳しくは [beta-privacy.md](beta-privacy.md) を確認してください。
