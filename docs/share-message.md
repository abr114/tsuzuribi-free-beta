# つづりび 無償β 共有用文面

つづりびの無償β準備版を公開しました。

- 本番URL: https://tsuzuribi-alpha.vercel.app/
- GitHub: https://github.com/abr114/tsuzuribi-free-beta

つづりびは、AIが慰めるためのアプリではなく、結果が出る前の努力や行動の跡をあとから見つけ直すための「自己証拠箱」です。

現在は、Googleカレンダー、今日のこと、メモ貼り付け、手紙、つらい時の画面から、短い証拠ラベルを保存前に確認して残せます。Googleを使わない場合も、今日のことやメモ貼り付けから試せます。

保存するものは、確認した短いラベル、カテゴリ、日付ラベル、ユーザー自身が入力した手紙本文です。Google予定タイトルの生データ、eventId、calendarId、場所、参加者、詳細本文、Google APIレスポンス全体は保存しません。現在の保存先は、この端末のブラウザ内です。

Settingsには、お問い合わせフォームへの導線があります。改善点、不具合、使ってみた感想、連絡したいことを送れます。実名、企業名、予定名などは書かなくて大丈夫です。

無償βでは、DB本接続、AI、通知、課金、Google裏側定期取得、GitHub/Notionなどの外部連携はまだ入れていません。まずは「残っていたことが迷わず見える体験」が成立するかを確認します。
